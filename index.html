SIMPLE-SPECS

Det lynhurtige, bærbare hardware-dashboard til Windows. Ingen installation, ingen reklamer, kun de rå data du har brug for.

Se Kildekoden på GitHub

ZERO INSTALL

Hele værktøjet er pakket i én enkelt lille .exe-fil. Kør den direkte fra dit skrivebord, et netværksdrev eller en USB-nøgle.

ÆGTE NØRD-DATA

Se SSD-modeller, RAM-producenter og korrekte MT/s hastigheder. Alt filtreres intelligent for at fjerne støj og OEM-fejl.

1-CLICK EKSPORT

Kopier lynhurtigt alle specifikationer og netværksinfo til udklipsholderen med ét klik. Perfekt til salgsannoncer og IT-support.

Gennemsigtighed & Sikkerhed

Simple-Specs er bygget af IT-folk til IT-folk. Vi hader oppustet software og skjult telemetri lige så meget som du gør.

Derfor aflæser programmet udelukkende dine hardwaredata lokalt via Windows WMI (Windows Management Instrumentation). Der sendes nul bytes tilbage til os. Vil du være helt sikker? Koden er 100% åben, så du kan læse den igennem eller bygge programmet selv fra bunden.
